¿Como se Elaboran los productos?

Actualmente se hacen estos pasos para la elaboracion de estos productos llamativos

1. Para el proceso de fabricación se usa una impresora 3D creality cr-10 v3.

Creality cr-10 v3

2. El material que usa esta impresora 3D para realizar Las impresiones es PLA de 1.75mm la cual se puede encontrar en una gama amplia de colores

Material de impresion PLA

3. Se crea el diseño en el programa Ultimaker cura y se pasa por USB a la impresora, este diseño también puede ser a petición del cliente.

Ultimaker cura

4. Para una figura de 10CM el tiempo de elaboración es de 6 horas

Ultimaker cura

5. Luego de su elaboración se debe lijar, pintar y dar acabados finales

Ultimaker cura
